# Objects and Records

## What is an Object?

An object is a database table in Salesforce used to store data.

Examples:
- Account
- Contact
- Opportunity

## Types of Objects

### Standard Objects
Pre-built Salesforce objects.

Examples:
- Account
- Contact
- Lead

### Custom Objects
Objects created by administrators to meet business requirements.

Example:
- Customer
- Support Ticket

## What is a Record?

A record is a single row of data stored in an object.

Example:

Object: Customer

Records:
- Rahul Sharma
- Priya Singh
- Amit Kumar

## Interview Question

Q: What is the difference between an Object and a Record?

A:
An Object is a database table, while a Record is a single entry stored within that table.

## Key Takeaway

Object = Table

Record = Row
