# Objects and Records

## What is an Object?

An Object is a database table in Salesforce used to store business data.

Examples:
- Account
- Contact
- Lead
- Opportunity

---

## Types of Objects

### Standard Objects

Pre-built objects provided by Salesforce.

Examples:
- Account
- Contact
- Lead
- Opportunity
- Case

### Custom Objects

Objects created by administrators to meet business requirements.

Examples:
- Customer
- Employee
- Support Ticket
- Asset Request

---

## What is a Record?

A Record is a single entry stored in an Object.

Example:

Object: Customer

Records:
- Rahul Sharma
- Priya Singh
- Amit Kumar

---

## Object vs Record

| Object | Record |
|----------|----------|
| Database Table | Row in Table |
| Stores data structure | Stores actual data |
| Example: Customer | Example: Rahul Sharma |

---

## Real-World Example

Object: Employee

Fields:
- Employee Name
- Employee ID
- Department

Record:

Employee Name: John Smith  
Employee ID: EMP001  
Department: IT

---

## Interview Question

### Q: What is the difference between an Object and a Record?

**Answer:**

An Object is a database table that stores a type of information, while a Record is a single row of data within that object.

---

## Key Takeaway

Object = Table

Record = Row

Field = Column
